# 人事評価・研修統合プラットフォーム企画書

## 1. はじめに

### 1.1. 背景と目的

一般社団法人人材育成支援機構様は、企業向けに人材育成の研修及び動画コンテンツを提供されています。本企画は、これらの研修サービスをさらに発展させ、人事評価制度と連携させた統合プラットフォームを構築することを目的とします。

現状の課題として、研修の成果が人事評価に直結しにくい、個々の従業員に最適な研修を提供できていない、といった点が挙げられます。本プラットフォームは、これらの課題を解決し、人材育成の効率化と効果の最大化を目指します。

### 1.2. コンセプト

**「評価」と「育成」をシームレスに繋ぐ、次世代型人材開発プラットフォーム**

本システムは、人事評価の結果を基に、個々の従業員に必要な研修を自動で推奨します。また、研修の受講状況やテスト結果を人事評価に反映させることで、客観的で納得感のある評価を実現します。これにより、従業員の自律的なスキルアップを促進し、組織全体のパフォーマンス向上に貢献します。

### 2.2. 主要機能一覧

| システム | 主要機能 |
|---|---|
| **人事評価システム** | 目標設定・管理、評価シート作成、評価ワークフロー、360度評価、評価結果分析 |
| **研修管理システム(LMS)** | 研修コンテンツ管理（動画、PDF、SCORM）、受講者管理、進捗管理、テスト・アンケート作成、レポート出力 |
| **連携機能** | 評価結果に基づく研修レコメend、研修履歴の評価への反映、スキルマップ連携 |

## 3. 各システムの詳細

### 3.1. 人事評価システム

従業員の目標設定から評価の実施、フィードバックまでを一元管理します。評価プロセスの透明性を高め、従業員の納得感を醸成します。

- **目標設定・評価機能**: MBO（目標管理制度）やOKR（Objectives and Key Results）など、様々な評価手法に対応可能です。
- **評価ワークフロー管理**: 評価者、被評価者、承認者など、役割に応じた権限設定とワークフローの自動化を実現します。
- **評価結果の分析・活用**: 評価結果を多角的に分析し、人材配置や育成計画の策定に活用します。

### 3.2. 研修管理システム（LMS）

動画コンテンツを中心とした多様な研修教材を管理・配信します。従業員は時間や場所を選ばずに学習を進めることができます。

- **研修コンテンツ管理**: 動画、PDF、PowerPoint、SCORM規格の教材など、様々な形式のコンテンツに対応します。
- **受講管理・進捗確認**: 管理者は、従業員一人ひとりの受講状況や進捗度をリアルタイムで把握できます。
- **テスト・アンケート機能**: 研修内容の理解度を確認するためのテストや、研修効果を測定するためのアンケートを簡単に作成・実施できます。

### 3.3. 連携機能

本プラットフォームの核となる、人事評価と研修の連携機能です。

- **人事評価結果に基づく研修レコメンド**: 評価結果で明らかになった課題や、次の等級に求められるスキルに基づき、推奨される研修コースを自動で提示します。
- **研修受講履歴の人事評価への反映**: 研修の受講履歴、テストの成績、取得した資格などを人事評価の参考情報として活用できます。
- **スキルマップと連動した育成プランの提示**: 従業員一人ひとりのスキルマップを作成し、目標とするキャリアパスの実現に向けた育成プランを可視化します。

## 4. システム構成案（技術スタック）

拡張性とメンテナンス性を考慮し、モダンな技術スタックを提案します。

| カテゴリ | 技術選定案 |
|---|---|
| **アーキテクチャ** | マイクロサービスアーキテクチャ |
| **フロントエンド** | Next.js (React) |
| **バックエンド** | Node.js (TypeScript), Go |
| **データベース** | PostgreSQL, Redis |
| **インフラ** | AWS (ECS, S3, CloudFront, RDS) |
| **LMS基盤** | オープンソースLMS（Moodle等）のカスタマイズ、またはフルスクラッチ開発 |

## 5. その他

### 5.1. セキュリティ

個人情報や評価情報など、機密性の高い情報を取り扱うため、多層的なセキュリティ対策を講じます。
- 通信の暗号化 (TLS 1.3)
- データベースの暗号化
- 多要素認証 (MFA)
- IPアドレス制限
- 定期的な脆弱性診断

### 5.2. 今後の展望

将来的には、AIを活用した以下のような機能拡張も視野に入れています。
- よりパーソナライズされた研修レコメンド
- キャリアパスのシミュレーション
- ハイパフォーマーの行動特性分析
